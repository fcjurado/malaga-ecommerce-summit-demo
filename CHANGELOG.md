# Changelog

All notable changes to this project will be documented in this file.

## 1.0.2 - 2026-03-19

- Added a minimal `.gitignore` for local development artifacts.
- Added `build-release.sh` to generate reproducible WordPress plugin ZIP packages.
- Added `CHANGELOG.md` as part of the distributable release contents.
- Standardized local release packaging through the `dist/` directory.

## 1.0.1 - 2026-03-19

- Exposed selected WordPress core abilities for MCP usage.
- Added WooCommerce demo abilities for orders, customers, coupons, refunds, and order notes.
- Added shared helpers for permissions, metadata, date validation, and response formatting.
- Added English README documentation with natural-language MCP prompt examples.
- Added release packaging workflow for ZIP distribution.
